# 나만의 작은 비밀 노트 (SecretNote)

하루하루 겪은 일, 느꼈던 감정을 일기처럼 작성하고, 저장해 놓는 게시판입니다.
<br/>

- 클라이언트 : React
- 서버 : Node Express
- 데이터베이스 : MongoDB
- 배포 : Firebase functions, Vercel

<br/>

## ✨ 프로젝트 시작하기

server, client

```bash

npm install

npm run start

```
